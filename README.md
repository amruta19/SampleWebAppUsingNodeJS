# SampleWebAppUsingNodeJS
Web application using Angular JS (UI) and Node JS (for Server/API)

Initially restore table into database called "test"

To start Server:
Download and install node js
Create folder add server.js and package.json to that folder
npm install to same folder
Then start server using command [node server.js]

Create other folder add script folder and html file to that folder
Then by clicking on html file appliction will be started
